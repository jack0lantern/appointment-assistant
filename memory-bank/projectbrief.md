# Project Brief: Appointment Assistant

## What This Is

**Appointment Assistant** is an AI-powered mental health treatment plan assistant. Therapists upload session transcripts; the system generates structured, evidence-based treatment plans with citations, a client-friendly view, and safety flag detection.

## Problem Solved

- Manual treatment plan documentation is time-consuming for therapists.
- Clinical language is inappropriate for client-facing summaries.
- Safety concerns (suicidal ideation, substance crisis, etc.) need systematic detection.

## Who It's For

- **Therapists** — Generate and edit treatment plans from session transcripts.
- **Clients** — View plain-language summaries of their treatment plans.

## Core Value

1. **Dual views** — Full clinical documentation for therapists; accessible summaries for clients.
2. **Citation-based explainability** — Line references back to the transcript for verification.
3. **Safety detection** — AI + regex layers for crisis/substance flags.
4. **Version immutability** — Audit trail for edits; no overwrites.
